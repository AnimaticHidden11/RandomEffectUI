Предыдущая версия была RandomEffects была BETA 1.0. 
И я выпустил более улуччшенную ее версию!

Changes
EN:
— Added a full—fledged config (config.yml)(after starting the server, you will have a new RandomEffectsUI folder in the plugins folder, and there will be a config.yml file. Come in and you can configure the plugin.)
— Fixed effect levels (now all from the config)
— Fixed the duration of useful effects
— The first effect is now at a preset interval
— Fixed level shifting (-1)
— Improved performance and stability
— Added English localization of the plugin (Now available only EN/RU)
— After entering the world, you will receive a chat command that allows you to activate random effects, which you can also turn off (the command prompt will be displayed)

Изменения
RU:
— Добавлен полноценный конфиг (config.yml)(после запуска сервера у вас появится в папке plugins, новая папка RandomEffectsUI, и там будет файл config.yml. Заходите, и можете настроить плагин) 
— Исправлены уровни эффектов (теперь всё из конфига)
— Исправлена длительность полезных эффектов
— Первый эффект теперь через заданный интервал
— Исправлено смещение уровней (-1)
— Повышена производительность и стабильность
— Добавлена английская локализация плагина (Доступно теперь только EN/RU)
